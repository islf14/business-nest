# BUSINESS DIRECTORY

- App to list, create, show, update and delete business.

- Uses NestJS, Typescript, React, Tailwind Css, Vite and PostgreSQL.

## Requirements

- Node.js must be installed.
- PostgreSQL

## How to run

Perform the following steps:

- To install dependencies, run in console:

```bash
npm install
```

- To build the project, run in console:

```bash

npm run build
```

- To start the project, run the following command:

```bash
npm run start
```

- To use, open in your browser (port set to env).

```bash
http://localhost:3000/
```
